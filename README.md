# 語

有五素：`列/連`，`列/空`，`文`，`構`，`集`。構五素以爲化，而化於宇宙內外。

## 列

`列`，有序也。

### 列/連

`列/連`者，連首於一列也。非Scheme之cons，故不可以爲pair。

### 列/空

空非無，空乃有。`列/空`爲Scheme之`'()`。

## 文

`文`，一物也。非句。`文`可以構爲`名`。`名`似Scheme之`symbol`。`string`可爲一`文`，`string->number`非一`文`。

## 構

`構`有一名，一`列`定長。

## 集

`集`，無序也。
構`列`爲`集`。`集`可有窮，亦可無垠。
`集`可反構之爲`列`而化之。
