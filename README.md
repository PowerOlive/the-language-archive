### 2.2. 述


## 3. 誤

### 3.1. 無

替換：在進行0或更多次替換後，可以把沒有結果（需要無限的時間的）的任意個`:物`替換爲`(→誤 (→構 {引 誤/界/無}
<未定義>))`，`<未定義>`是任意的`:物`。

實現應該儘量避免無限的運行下去。

實現應該儘量避免無用的替換。

### 3.2. 化

內置的任何`:化`產生的`:誤`是`(→誤 (->:構 {引 誤/界/化} ((→化 '(:列) ':列) <:名> <式>
<位:名>)))`， `<:名>`是它的名稱，`<式>`是它的參數，`<位:名>`是產生`:誤`的參數的名稱。
