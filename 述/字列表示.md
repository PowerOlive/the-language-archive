字列表示
======
+ 名 `名(<x> ...)`
+ 列 `(<x> ...)` `()` `(<a> ... - <d>)`
+ 字 `字<x>` 例 `字甲`
+ 循環 `周<x>` `周<x>(<v>)` 例 `周戊戌((字甲 - 周戊戌))` 是 Scheme的`#0=(#\甲 . #0#)`
+ 常量 `空` `陰` `陽`
+ 映 `映((<名> <物>) ...)`
+ 機 `機(<形> <物>)`
+ 引機 `引機(<形> <物>)`
+ 譯機 `譯機(<形> <物>)`
+ 誤 `誤(<x>)`
+ 構 `構((<x> ...) (<物> ...))` `名(<x> ...)`是它的名
